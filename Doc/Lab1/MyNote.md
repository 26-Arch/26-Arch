## Instruction（指令）

### 基础概念

**ISA**: Instruction Set Architecture  
指令集架构  

代表性指令集架构：

1. x86: 复杂指令集计算机  
2. ARM: 精简指令集计算机  
3. RISC-V: 精简指令集计算机  

我们平时电脑上所见的一般为 x86（Intel/AMD）架构  
或者 ARM（Apple）架构  
移动平台上一般是 ARM 架构  

指令集架构的意义：  
作为接口将软件与硬件进行连接  
在使硬件实现对软件透明（即不可见）的前提下，为软件提供运行的规范与约束  
同时也是硬件实现的目标与方向  

### RISC-V

**RISC-V**: Reduced Instruction Set Computer - Five  
第五代精简指令集计算机  

#### 设计理念：**模块化**  

1. **Base ISA**（基础指令集）：  
    核心：**RV32I**（32位）或 **RV64I**（64位）基础整数指令集  
    这是基础，保持不变，仅含40余条最基础指令，足以运行完整软件
2. **Standard Extension**（标准扩展）：  
    根据需求，硬件可以选择性添加扩展模块，常见：
    - **M**：整数乘除法扩展
    - **A**：原子操作扩展（在多核场景下用于同步）
    - **F/D**：单精度/双精度浮点数扩展
    - **G**：通用组合（IMAFD），被视为通过用处理器的标准配置 // TODO: 没懂
    - **C**：压缩指令扩展，将常用指令压缩为16位以缩减代码体积
    - **V**：向量扩展，用于处理高性能计算中的数据级并行

#### 特性

1. **寄存器组织**：  
    32个通用寄存器（x0-x31），**x0 应连线为常数 0** // TODO:什么意思，什么作用？
2. **指令格式规整**：
    仅含 6 种基本指令格式，且为 32 位定长，降低了译码器复杂度  
    基本指令格式：  
    - **R**：
        - 寄存器型，用于 **R-R** 操作  
        - 操作数（字段特点）：两个源寄存器（rs1, rs2），一个目标寄存器（rd）
        - 示例：ADD, SUB, AND, OR, XOR, etc.
    - **I**：  
        - 立即数型，用于 **短立即数** 运算、**Load** 访存指令、**JALR** 跳转指令
        - 操作数（字段特点）：一个源寄存器（rs1），一个12位立即数（imm），一个目标寄存器（rd）
        - 示例：ADDI（加立即数），ANDI（与立即数），LW（加载字），LB（加载字节），JALR（跳转并链接），etc.
    - **S**：
        - 存储型，用于 **Store** 指令，将寄存器数据写入内存
        - 操作数（字段特点）：两个源寄存器（rs1 基址，rs2 数据），**立即数被拆分** 于两个字段中，以保证 rs1 和 rs2 位置与 **R** 型一致  
        - 示例：SB（存储字节），SH（存储半字），SW（存储字），etc.
    - **B**：  
        - 分支型，用于 **条件分支** 跳转，地址相对于当前 PC
        - 操作数（字段特点）：目的寄存器（rd）和 20 位高位立即数
        - 示例：LUI（加载高位立即数），AUIPC（PC 加上高位立即数），etc.
    - **J**：
        - 跳转型，**无条件直接跳转**
        - 操作数（字段特点）：目的寄存器（rd，通常存返回地址），20 位打乱顺序的立即数
        - 示例：JAL（跳转并链接），etc.

3. **寻址方式简化**：  
    仅支持 **寄存器相对寻址**（寄存器 + 12位偏移量）和 **立即数寻址**  
    *与之相对，x86 有多种寻址方式，如直接寻址、间接寻址、基址寻址、变址寻址等*  
4. **存储器模型**：  
    **小段字节序**（Little-Endian），目前的主流选择  

### 五级流水线 CPU

**五级流水线**：  

1. **IF**(Instruction Fetch): 取指
2. **ID**(Instruction Decode): 译码
3. **EX**(Execute): 执行
4. **MEM**(Memory): 访存
5. **WB**(Write Back): 写回
